= bio-dbla-classifier

DBL-alpha tags can be classified into six groups depending on the number of cysteines and presenced of 
sequence motifs REY and MFK. This plugin extends the Bio::Sequence::AA class by adding methods for grouping the tags.

= Installation
 
 gem install bio-dbla-classifier

= Uninstall
 
 gem uninstall bio-dbla-classifier
